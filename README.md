# [unixzii.github.io](https://unixzii.github.io)

Personal blog.

## Build

Setup the project:

```shell
git clone https://github.com/unixzii/unixzii.github.io-src.git
cd unixzii.github.io-src
npm install
```

Build and serve:

```shell
rollup -c
serve -s
```

## Deploy

The built-in deploy script is currently for owner. Don't run it.
